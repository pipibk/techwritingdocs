Build a Basic Blockchain in Python

Estimated practice time: 45 minutes | Skill level: Intermediate/Advanced | Written by: Rachael O. Afolabi, 2025

Overview

This codelab guides you through building a simple blockchain in Python, step by step, with no external libraries required. You will learn how blocks are structured, how cryptographic hashes work, and how data is linked securely to form a chain. This codelab is designed for intermediate Python learners, and focuses on core blockchain principles—like immutability and integrity. By the end, you will have a functional blockchain and a clear understanding of how real-world blockchain systems are built from the ground up.

Environment - This codelab uses VS Code as the code editor and IDE. The instructions can work with other code editors.

Step 1: Set up your environment​

Ensure that Python is installed on your computer. If it is not, download it here Python and follow the installation instructions.

tip

To check if Python is installed on your computer, you can run python3 on your terminal.

Step 2: Define a Block​

Create a file called blockchain.py. Start by defining the structure of a block.

class Block:
    def __init__(self, index, timestamp, data, previous_hash):
        self.index = index
        self.timestamp = timestamp
        self.data = data
        self.previous_hash = previous_hash
        self.hash = self.calculate_hash()

    def calculate_hash(self):
        block_content = f"{self.index}{self.timestamp}{self.data}{self.previous_hash}"
        return hashlib.sha256(block_content.encode()).hexdigest()
breakdown and definitions

Step Breakdown: This step defines a block with attributes for indexing, time, data, and the previous hash. Generates a secure hash using SHA-256 based on block contents."

Block​

  • Definition: A single unit of data in a blockchain, like a container that holds information.
  • Analogy: Think of a block like a page in a notebook. Each page stores a piece of information, and flipping through the pages shows the full story.

Index​

  • Definition: A number that shows the block’s position in the chain.
  • Analogy: Like page numbers in a book — they help keep blocks in the correct order.

Timestamp​

  • Definition: The exact date and time when the block was created.
  • Analogy: Similar to a receipt timestamp — it tells you exactly when a transaction or event happened.

Data​

  • Definition: The actual content inside the block (e.g., transaction info).
  • Analogy: Like the text or drawing on each notebook page — it’s the meaningful stuff you're storing.

Previous Hash​

  • Definition: A reference to the hash of the previous block in the chain.
  • Analogy: Like a linked chain of safes, where each safe contains the key to the one before it. If one safe is tampered with, all the safes after it become suspicious.

Hash​

  • Definition: A unique digital fingerprint for each block, generated from its content.
  • Analogy: Like a DNA sample — if even one tiny piece of the block changes, the DNA (hash) completely changes too.

SHA-256 (Secure Hash Algorithm)​

  • Definition: A cryptographic function that converts data into a fixed, secure 64-character string.
  • Analogy: Like a high-tech paper shredder that always shreds into the same pattern for the same input — but there's no way to reverse the process and rebuild the paper.

Step 3: Create the Genesis Block​

This is the first block in your chain.

def create_genesis_block():
    return Block(0, str(datetime.datetime.now()), "Genesis Block", "0")
What this means

Creates a special block with no previous block, using "0" as the previous hash.

Step 4: Add New Blocks​

Write a function that builds new blocks on top of the existing one.

def next_block(last_block, data="New Data"):
    index = last_block.index + 1
    timestamp = str(datetime.datetime.now())
    previous_hash = last_block.hash
    return Block(index, timestamp, data, previous_hash)

Step 5: Build the Blockchain​

Now you can link blocks together in a chain.

blockchain = [create_genesis_block()]
num_blocks_to_add = 5

for i in range(num_blocks_to_add):
    new_block = next_block(blockchain[-1], f"Block #{i + 1} Data")
    blockchain.append(new_block)
    print(f"\n--- Block {new_block.index} ---")
    print(f"Timestamp: {new_block.timestamp}")
    print(f"Data: {new_block.data}")
    print(f"Previous Hash: {new_block.previous_hash}")
    print(f"Current Hash: {new_block.hash}")

Step 6: Run the Code​

Open the terminal and run this command:

python3 blockchain.py
