Getting Started with Power BI Web

Estimated practice time: 30 minutes | Skill level: Beginner | Written by: Rachael O. Afolabi, 2025

Overview

This guide introduces beginners to Power BI Web (Power BI Service). In this tutorial, you will learn how to upload a dataset, create visualizations, and share reports — all from your browser, no downloads required.

Environment - This tutorial uses Microsoft's Power BI Service which is accessed on the browser. You will need a school or work email address to access this service.

What is Power BI Web?​

Power BI Web is a cloud-based version of Power BI, accessible through your browser. It lets you:

  • Upload data (CSV, Excel)
  • Create interactive dashboards
  • Share reports online
Access

Access Power BI Web here: https://app.powerbi.com

Create a Free Power BI Account​

  1. Go to https://powerbi.microsoft.com
  2. Click Sign Up Free
Take Note

Use a work or school email to sign up (personal accounts like Gmail may not work). Follow the steps to verify and access the platform.

Upload Your First Dataset​

Sample Dataset: Download Power BI Sample Dataset​

You can right-click > Save As to download it.

Steps to Upload Dataset​

  1. Log into https://app.powerbi.com
  2. Click Create in the left-hand menu
  3. Choose Upload a file
  4. Select Local file
  5. Upload your dataset and click Create Report

Create a Simple Report​

  1. In the report editor:
    • Drag category or state to the X-axis
    • Drag total exports to the Y-axis
    • Select Bar Chart or any other visual
  2. Use the Visualizations pane to change the chart type
  3. Customize:
    • Add a chart title
    • Change colors and labels in the Format panel

Add Filters and Slicers​

  • Drag a field like category to the Filters pane
  • Add a Slicer to filter by state or category
  • Try interacting with multiple visuals — they sync automatically

Save and Share Your Report​

  • Click File > Save, give your report a name
  • To share:
    1. Click Share in the top menu
    2. Invite others via email (they must be part of your organization) or publish to web (admin permission is needed) for this action.
